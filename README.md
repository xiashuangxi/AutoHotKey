# AutoHotKey

我的 Autohotkey 脚本，是为了将键盘按键操作控制在小范围内，减少手臂频繁移动和敲按键的次数，具体的使用方法请参考 [Wiki](https://github.com/xiashuangxi/AutoHotKey/wiki)。
