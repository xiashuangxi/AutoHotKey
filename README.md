# AutoHotKey

这是我个人提升生产力的Autohotkey套件。

## 快捷键绑定

*这是笔记本键盘布局*

<img src="autohotkey.svg" alt="快捷键绑定" />

### Cmd

* <kbd>Ctrl</kbd>+<kbd>v</kbd>：粘贴
* <kbd>Ctrl</kbd>+<kbd>v</kbd>：清除屏幕

## 使用

* 下载安装[Authotkey](https://autohotkey.com/)
* `git clone https://github.com/xiashuangxi/AutoHotkey.git`
* 运行`Main.ahk`
